# Python scripts for Revit

#### FR

Ces fichiers sont utilisé via pyrevit. Pour les utiliser via RevitPythonShell supprimer ces lignes :
```
__doc__ =
__title__ =
__author__ =
```
Avant de lancer le script il faut ouvrir un fichier excel dans lequel le quantitatif sera écrit.
Le programme peut-êre améliorer car pour l'instant il ne marche pas si il n'y a pas au moins une longueur droite, un raccord et un accessoire par circuit, ce qui sera le cas dans la plupart des installations, sauf peut-être pour les canalisations.
